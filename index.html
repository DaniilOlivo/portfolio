Веб-сайт базируется на фреймворке языка Python – Django, с помощью которого реализуется маршрутизация, шаблонизация, административная панель и взаимодействие с базой данных. В качестве базы данных был выбран классический вариант – MySQL. В базе хранятся данные об новостях и товарах, в том числе и ссылки на изображения, которые хранятся прямо в файловой системе хостинга. В будущем, при расширении компании, планируются перейти на облачные файловые сервера.

Клиентская часть сайта выполнена на классическом стэке HTML, CSS и JavaScript. Поскольку у сайта, ввиду его специфики, немного динамики, большая часть интерактивных элементов сделана на чистом JS.

Сайт лендинг "Мульти-Сувенир"

Коммерческий проект, сайт лендинг компании «МультиСувенир», специализирующиеся на производстве пластмассовых игрушек.

На сайте располагается каталог товаров, контактная информация, а также новостная лента, которая может быть дополнена персоналом, благодаря административной панели сайта. Панель админа также дает возможность добавлять новые товары в каталог и редактировать права пользователей.

Проект имеет поддержку мобильных устройств, подстраиваясь под небольшие экраны телефонов и планшетов

Главная страница дает возможность начать рисовать, выдавая случайный референс по определенному алгоритму. Наиболее часто она будет давать работы, где были допущены ошибки и с проблемными тегами. После работы над референсом, ему выдается статус, который и берется за основу вычисления приоритета выдачи этого изображения. На странице статистики можно подробно увидеть сколько каких изображений есть в библиотеки пользователя в целом и по каждому отдельному тегу

В будущем планируется добавить подробную и удобную страницу для обзора референсов, а также улучшить систему выдачи изображений, добавив больше возможностей для настроек.

Подробности

Игра использует Socket.io для обновления игры, а также управлением комнатами, где собираются игроки. Каждый желающий может создать комнату, которую увидят другие, получая возможность подключится. Для старта игры нужно минимум двое, после чего начинается стандартная монополия. Игроки кидают кубики и ходят по игровому полю, покупая собственность, которой они могут обмениваться.

В качестве фротенда был выбран фреймворк Vue, поскольку требовалось обновлять и проверять множество информации на клиенте. Vue связывается с сервером через веб сокет, посредством socket.io, по которому постоянно получает состояние игры, ее ядро.

Сама реализация и состояние, каждой игры, что привязывается к комнате, хранится на сервере под управлением Express на Node.js. Сервер берет на себя менеджмент соединений, из идентификацию с соответствующей комнатой, а также управлением состояние игры. Как только состояние игры обновляется, то сервер рассылает всем ее состояние, обновляя состояние приложения Vue на каждом клиенте.

Монополия

Своя собственная реализация известной игры монополии, которая позволяющая вам играть с другими людьми по сети. Сервис использует технологию веб сокетов, что позволяет ему обновляться в режиме реального времени.

За основу взяты оригинальные правила и карточки собственности. Кроме самой игры присутствует чат, который также берет на себя роль логов, записывая все события, что происходят в игре.
